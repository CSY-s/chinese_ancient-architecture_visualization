# 中国古代建筑文化可视化展廊

基于 Vue 3 + ECharts 构建的中国古代建筑文化展示项目。

## 技术栈
    Vue 3 (Composition API)
    Vite
    ECharts 5
    echarts-wordcloud

## 可视化内容

- **雷达图**：五大建筑流派（徽派/晋派/苏派/岭南/川派）多维度文化价值对比
- **柱状图**：建筑热度排名 & 游客打卡热度
- **词云**：营造相关的历史文献关键词频次
- **关系图**：建筑与非遗、著名匠师/文献的关联网络
- **折线图**：建筑功能演变（宗教→居住→公共→文旅）
- **饼图**：艺术评级分布
